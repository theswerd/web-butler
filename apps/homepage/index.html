An actual errand: “always hide this” becomes a page extension that holds on every future visit. The controls are real, too: try the crosshair and pick an element anywhere on this page.

The service

  • Answers, in place about any page

    Ask about the page you’re on. Pick elements to point at; the butler reads exactly what you picked and replies right there.

  • Errands, run by a visible hand

    Forms filled, flows walked, drafts written: a cursor moves through your tab while you watch it work.

  • Alterations, kept changes that stick

    “Always hide this.” Saved as tidy page extensions that reapply on every matching visit, and mend themselves when a site shifts.

  • Reports, delivered filed in the side panel

    Longer research and write-ups arrive as documents in Chrome’s side panel, kept on file to reopen any time.

The arrangement

No new subscription. Web Butler signs in with the ChatGPT, Claude, or Grok plan you already pay for, and every request runs in your own private sandbox against your own account. Open source, end to end.

Getting started

  1. Install

    Clone theswerd/web-butler and load the unpacked build. Chrome Web Store to follow.

  2. Connect

    Sign in once with ChatGPT, Claude, or Grok. The butler keeps your seat.

  3. Summon

    Press ⌘E on any page. The bowtie appears; ask away.
